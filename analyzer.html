About

Purpose of This Website

This platform allows users to seamlessly upload structured CSV datasets, automatically clean messy data, perform comprehensive Exploratory Data Analysis (EDA), visualize hidden patterns, and prepare machine-learning-ready datasets entirely within the browser.

Data Preprocessing Steps

  • Missing Value Handling: Completely empty rows are automatically detected and removed from the dataset.
  • ND Value Replacement: Invalid entries such as "ND" or empty strings in numeric columns are safely converted to 0.
  • Numeric Conversion: Features are cast to strict numeric types to prevent computational errors.
  • Min-Max Normalization: Optional scaling compresses numerical features between 0 and 1, ensuring uniform feature weights for ML models.

Exploratory Data Analysis (EDA)

  • Dashboard Metrics: Displays the Highest Value, Average, Volatility (Standard Deviation), and overall trajectory direction.
  • Trend Analysis Chart: Visualizes the temporal or sequential progression of primary and comparative features over a selected row range.
  • Value Distribution: A dynamic histogram grouping feature values into frequency bins to analyze data spread.
  • Correlation Matrix: Visualizes positive and negative correlations across variables to identify feature relationships.

Machine Learning Simulation

The platform simulates forecasting behavior inside the browser using various algorithmic models:

  • Ridge Regression: Simulates L2 regularization to prevent overfitting by penalizing large weights.
  • Lasso Regression: Simulates L1 regularization for feature selection logic, leading to simpler prediction tracks.
  • K-Nearest Neighbors (KNN): Simulates local similarity-based predictions resulting in stepped forecasts.
  • Support Vector Regression (SVR): Simulates margin-based forecasting.

Machine Unlearning

Inspired by regulatory compliance requirements, this feature uses robust Z-Score anomaly detection to identify extreme, highly volatile outliers. These influential points are actively removed ("forgotten") from the model's memory map to prevent data poisoning bias, providing a frontend simulation of Certified Unlearning.

Final Output

The fully processed and normalized dataset can be exported as a clean CSV file. This standardized output is ready for immediate use in ML training on external platforms like Python (Pandas/Scikit-learn) or TensorFlow.
